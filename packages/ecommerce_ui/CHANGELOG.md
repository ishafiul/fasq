# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release
- Complete theme system with light/dark mode support
- 30+ pre-built widgets
- 200+ SVG icons (filled, outlined, twotone)
- Typography system with Golden Ratio scale
- 8-point grid spacing system
- 4-point grid radius system
- Semantic color palette
- Arial font family included
